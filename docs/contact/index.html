Contact GRA Release Radar

GRA Release Radar is maintained through its public GitHub repository. The most reliable contact path is to open a GitHub issue for bug reports, documentation corrections, feature requests, or questions about how the release analytics are calculated. Public issues also help other maintainers see known limitations and proposed improvements.

Repository: github.com/Caldis/github-releases-downloads-analysis.

Issues: github.com/Caldis/github-releases-downloads-analysis/issues.

When reporting a problem, include the repository you analyzed, whether you used a GitHub token, the visible error message, and whether the issue appears only for one repository or across multiple repositories. Do not paste private tokens into issues, screenshots, logs, or browser console output.
