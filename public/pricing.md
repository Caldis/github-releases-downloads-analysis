# GRA Release Radar pricing

Last updated: 2026-05-09

GRA Release Radar is currently free to use.

| Plan | Price | Best for | Included capabilities | Limits |
| --- | ---: | --- | --- | --- |
| Free client-only | $0 | Open source maintainers, release engineers, developer relations teams, and project evaluators | Public GitHub release analytics, release asset download totals, release cadence, packaging breakdowns, local browser cache, optional local GitHub token use | GitHub REST API limits apply. Unauthenticated requests share the user's IP quota. Authenticated requests use the user's GitHub account quota. |

There is no paid hosted plan, account system, billing page, seat pricing, usage-based metering, or enterprise contract for GRA Release Radar at this time.

The app does not charge for analysis and does not sell a hosted data export. Users are responsible for complying with GitHub's API terms and for protecting any GitHub personal access token they choose to use in their own browser.

If a user needs a server-side API, scheduled reports, team accounts, webhooks, or an MCP tool endpoint, those are not part of the current public product.
